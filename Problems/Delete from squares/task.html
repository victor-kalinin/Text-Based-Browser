Delete from squares

We have a dictionary squares which contains numbers as keys and their squares as values. It looks like this:

squares = {1: 1, 3: 9, 5: 25, ...}

Your task is to read from the input the value of the key that needs to be deleted from the dictionary and print the value of this key. If the number read from the input does not exist in the dictionary, then you need to print There is no such key.

The dictionary squares is already defined for you.


Sample Input:
4

Sample Output:
There is no such key



Memory limit: 256 MB
Time limit: 15 seconds

Caution

You may see errors in your code or in execution results because of missing context. Don't mind those; simply write your solution and click Check.
